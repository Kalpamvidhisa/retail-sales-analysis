# 🛒 Retail Sales Data Analysis (End-to-End Capstone)

## 📌 Project Overview

This project analyzes retail sales data to uncover meaningful insights and support data-driven decision-making.
It includes data cleaning, exploratory data analysis (EDA), and visualization to identify trends and patterns.

---

## 🎯 Objective

* Perform data cleaning and preprocessing
* Conduct exploratory data analysis (EDA)
* Create visualizations to understand sales trends
* Generate actionable business insights

---

## 📂 Dataset

* **Dataset Used:** Superstore Sales Dataset
* Contains information about:

  * Orders
  * Sales and Profit
  * Categories and Regions
  * Order Dates

---

## 🛠️ Tools & Technologies

* Python 🐍
* Pandas (Data Analysis)
* Matplotlib (Data Visualization)
* Jupyter Notebook

---

## 🔍 Steps Performed

### 1. Data Cleaning

* Removed missing values
* Eliminated duplicate records
* Converted date columns into proper format

### 2. Exploratory Data Analysis (EDA)

* Analyzed sales by category
* Examined monthly sales trends
* Evaluated profit by region

### 3. Data Visualization

* Bar chart: Sales by Category
* Line chart: Monthly Sales Trend
* Bar chart: Profit by Region (optional)

---

## 📊 Key Visualizations

* 📌 Sales by Category
* 📌 Monthly Sales Trend

---

## 🧠 Key Insights

1. The Technology category generates the highest sales → Focus more on this category
2. Sales increase during festive months → Plan seasonal marketing strategies
3. Some regions have lower profit → Optimize operational costs

---

## 📌 Conclusion

This project demonstrates how data analysis can help businesses identify trends and improve decision-making.
By focusing on high-performing areas and optimizing weak regions, companies can increase profitability.
